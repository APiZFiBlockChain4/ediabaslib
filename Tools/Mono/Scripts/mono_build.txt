Mono 4.0.1
Install mono with: msiexec -i xxx.msi INSTALLFOLDER="C:\Programs\Mono"

Install cygwin:
Add all packages from:
http://www.mono-project.com/docs/compiling-mono/windows/
Additionally:
- gettext
- zlib zlib-devel
- dos2unix
- pkg-config

extract:
tar -jxvf mono-xxx.tar.bz2

build (Path to existing mono):
export PATH=$PATH:/cygdrive/c/Programs/Mono/bin/
./configure --host=i686-pc-mingw32 --prefix=/cygdrive/c/Programs/MonoXXX/
make
make install
build MonoPosixHelper.dll with VS
- open msvc/mono.sln
- Compile monoposixhelper (debug only, set runtime of eglib and monoposixhelper to multitread-DLL (/MD))
- Copy MonoPosixHelper.dll to /cygdrive/c/Programs/MonoXXX/bin
