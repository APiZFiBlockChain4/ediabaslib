Howto install the Android application:
- Copy the .apk from the Apk directory to your device.
- Allow installation from unknown sources.
- With ES file explorer select the copied .apk file to install it.

The sample configuration in the Config directory only fits to my car (E61),
you have to adapt it as needed!

Install the sample configuration with the following steps:
- Copy the contents of the Config directory to the (external) media of the device.
- Additionally add all Ecu files from your EDIABAS/Ecu directory to the Ecu folder.
- After starting the application select the configuration
  E61Bt.cccfg for bluetooth or E61Enet.cccfg for ethernet communication.
